# 05. Контроль Качества (QC)

## 1. Типы Проверок
1.  **Consistency Check (Global):**
    *   Проверяет, не противоречит ли текст раздела "Паспорту исследования" (например, в тексте "Phase 2", а в глобалах "Phase 3").
2.  **Source Verification (LLM Judge):**
    *   Специальный агент сверяет сгенерированный текст с исходным Markdown'ом секции.
    *   Задача: "Найди галлюцинации в цифрах".

## 2. UI
*   Подсветка несоответствий (красным/желтым).
*   При клике на ошибку открывается "Diff" с исходником.

## 3. Workflow
*   QC запускается автоматически после генерации черновика AI.
*   Пользователь должен разрешить (Resolve) все критические замечания перед экспортом.
