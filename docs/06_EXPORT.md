# 06. Экспорт и Форматирование

## 1. Форматы
*   **DOCX:** Сборка документа из HTML (Tiptap) с применением корпоративных стилей.

## 2. Особенности экспорта таблиц
*   Поскольку мы парсим и генерируем таблицы в Markdown/HTML, при экспорте в Word они должны становиться "живыми" таблицами Word (не картинками).
*   Используем библиотеку `python-docx` (в Python сервисе) или `docx` (в Node.js) для корректного рендеринга границ и шрифтов.

## 3. Audit Log Export
*   Вместе с документом можно выгрузить "Audit Trail": CSV файл, где для каждого раздела указано, какой раздел протокола был источником и какой промпт использовался.
